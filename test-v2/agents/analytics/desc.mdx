# analytics

Agent d'analyse de données

## Informations

- **Type**: `test-v2/analytics`
- **Auteur**: Data Science Team
- **Version**: 1.0.0
- **Architecture**: Directive v2.0 (simplifiée)

## Description

Cet agent fait partie du projet **test-v2** et utilise la nouvelle architecture Directive v2.0 avec une structure simplifiée.

### Fonctionnalités

- ✅ Machine XState pour la logique métier
- ✅ Structure simplifiée `agents/analytics/`
- ✅ Base de données globale `~/.directive/data/`
- ✅ Type d'agent: `test-v2/analytics`

## Usage

### Déploiement

```bash
directive deploy agent analytics
```

### Test via API

```bash
# Démarrer le serveur
directive start

# Créer une session
curl -X POST http://localhost:3000/sessions \
  -H "Content-Type: application/json" \
  -d '{"agent_type": "test-v2/analytics"}'

# Lister les agents
curl http://localhost:3000/agents
```

## Développement

### Modifier la logique

Éditez le fichier `agent.ts` pour personnaliser la machine XState :

```typescript
export const analyticsMachine = createMachine({
  // Votre logique ici
});
```

### États disponibles

- **idle**: État initial, en attente
- **processing**: Traitement en cours  
- **success**: Traitement réussi
- **error**: Erreur rencontrée
